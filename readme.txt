Pomogaka is an ukagaka implementing the pomodoro method. 

https://en.wikipedia.org/wiki/Pomodoro_Technique
https://lifehacker.com/productivity-101-a-primer-to-the-pomodoro-technique-1598992730

Features include:
	Three built in timer presets and two custom presets; with an option to start a custom timer without saving it
	Continuous Mode, where pomogaka runs timers continuously without user input
	Long sessions, where you can queue a number of work sessions in a row
	Optional SFX when a timer ends
	Statistics for how many sessions worked and how many minutes worked
	Broadcasting statistics to other ukagaka open
	Static or visual timer shells
	Ability for users to make shells for pomogaka

Hotkeys (when pomogaka is selected):
	F1: Open this readme
The following hotkeys begin a work session without a goal immediately if a work session or break is not already running
	F5: 25/5
	F6: 50/10
	F7: 100/20
	F8: Custom preset one
	F9: Custom preset two
